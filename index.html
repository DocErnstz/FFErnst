Acerca de mi

Hola! mi nombre es Ernesto, soy un UI developer con 1 año y medio de experiencia creando cosas que viven en el internet, mi intereses en desarrollo web empezo en el 2019 cuando me encontre con un video de un chico creando animaciones de css que me asombraron. debido a este evento, me plantee construir paginas web que sean tan geniales como las de el usando HTML & CSS!

Exitosamente Construi y diseñe una pagina web usando el stack Sass-ReactJS-MongoDB para el desarrollo y AdobeXD para el diseño, la pagina que cree permitia a los clientes ver info del negocio y contactar al proveedor via mail desde el formulario de la pagina, durante alrededor de medio año estuve corrigiendo bugs y errores de diseño acorde al cliente. -Desarrolle de manera exitosa una landing page y posteriormente un proyecto multi-site SPA de 7 paginas usando Reactjs y Sass para el mismo cliente, lleve a cabo la construccion de varios wireframes y prototipos adaptados a diferentes dispositivos en Figma para el acuerdo del diseño de la landing y el multi-site para de esa manera iniciar el posterior proceso de desarrollo, realice mantenimiento de las dos websapp arreglando bugs de estilos css que pudieran surgir en algunos dispositivos mobiles además de realizar modificaciones de acuerdo a las necesidades del cliente.
